### Install

linux:
```shell
sudo apt install mosh tmux zsh autojump 
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```

mac: 
```shell
brew install mosh tmux autojumps 
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
